# Etapa Ads — Master Plan & Brief Index

**Updated:** 17 April 2026
**Structure:** flat, numeric-prefix filenames in posting order. 34 briefs cover the full 10-week `instagram-plan.md` schedule. Open a brief, copy each section into the matching ad-tool form field, attach the named brand-kit assets, generate.

Before running any brief through the tool, read in this order:

1. [`BRIEF.md`](./BRIEF.md) — why this setup exists
2. [`brand-assets-guide.md`](./brand-assets-guide.md) — the brand-kit PNGs (already generated in `_library/`) + avatar → coach mapping
3. [`instagram-plan.md`](./instagram-plan.md) — the 10-week posting cadence + brand-repetition rhythm

**Mix:** 24 blog-backed briefs (positions matching their topic posts) + 10 generic brand/product briefs (brand intro, 3× re-intros, 3× stat cards, 6× coach spotlights, launch CTA). Aggregate split is ~65% topic-educational / ~35% brand-product, which is the right ratio for a young brand on Instagram.

**Status key:** `brief-ready` · `generated` · `posted` · `blocked`

## Blocked / open items

- **Stat verification.** Briefs `08`, `14`, `25` all contain placeholder stats. Verify with Sport England, British Cycling, or academic sources and swap the `[XX]%` / `[X] IN [Y]` placeholders in each brief before shipping. Include the source line on the creative.
- **Brand kit PNGs.** All six already generated in `_library/` via `_library/regen.py`. Upload to the ad tool's Assets library once.
- **App screenshots for Avatar briefs.** Optional for Phase 1. If available, drop `app-plan-overview.png`, `app-coach-chat.png`, `app-week-view.png` into `_library/` and swap into the Media slot of any Avatar brief.
- **Landing banner on blog posts.** Every blog-backed brief's destination should greet the user with a small "You arrived from an AI-generated ad → [transparency page]" banner. Build the banner before the first ad goes live.
- **`.vscode/mcp.json` API keys.** Firecrawl + Perplexity keys are still in plaintext. Rotate and env-var them before the next push.
