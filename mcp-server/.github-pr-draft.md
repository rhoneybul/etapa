### PR body

```markdown
## Summary

Adds **Etapa** to the community servers list.

Etapa is an AI cycling coach for beginners and returning riders. This MCP exposes two tools:

- **`generate_training_plan`** — produces a personalised 2-4 week cycling training plan tailored to the rider's fitness level, goal, and available days. Calls the Etapa API under the hood (transparent about this in every response).
- **`cycling_beginner_guide`** — returns curated, jargon-free guidance across 9 topics (first bike, essential gear, first ride, nutrition, safety, habit-building, bike fit, and common mistakes). No API call, local content.

## Repo + links

- Source: https://github.com/rhoneybul/etapa/tree/main/mcp-server
- npm: https://www.npmjs.com/package/etapa-mcp (`npx -y etapa-mcp`)
- Hosted HTTP endpoint: https://etapa-mcp-production.up.railway.app/mcp
- Website: https://getetapa.com

## Transport

Both stdio (via npm/npx) and Streamable HTTP (hosted on Railway) are supported.

## Why it's useful

Most cycling training apps assume the user already understands training language (FTP, intervals, polarised plans, etc.). Etapa is explicitly designed for beginners, women getting into cycling, and returning riders — so the MCP speaks in plain English, never uses jargon, and caps plans at 4 weeks to stay honest about being a preview of the full app experience.

## Tested with

- Claude Desktop (stdio via npm)
- Claude Desktop (HTTP via mcp-remote)
- curl (HTTP protocol handshake + tool calls)

Happy to adjust the entry format or placement — thanks for maintaining this list!
```
